# Tabbycat Importer

Herramienta web para importar instituciones, equipos y jueges a torneos de debate en Tabbycat mediante API.

## 🚀 Uso

1. Abre [la aplicación]([https://TU-USUARIO.github.io/tabbycat-importer/app/](https://rgrandia.github.io/tabbyimporter/))
2. Configura tu conexión a Tabbycat
3. Sube tu archivo Excel siguiendo la plantilla
4. Verifica y importa

## 📋 Estructura del Excel

- **INSTITUTIONS**: Nombre, Código, Región
- **TEAMS**: Equipo, Institución, Participantes (x2) con emails y categorías
- **ADJ**: Nombre, Institución, Puntuación, Email

## 🔒 Seguridad

- Sin servidor intermediario
- API keys solo en tu navegador
- Código abierto y transparente

## 🛠️ Desarrollo

Tecnologías: HTML5, CSS3, JavaScript vanilla, SheetJS (XLSX)
